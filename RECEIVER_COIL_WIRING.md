# FDEM 接收线圈到 PXIe-4309 的接线方案

## 1. 适用范围

本文档适用于以下配置：

- 接收器件：线框式大接收线圈；
- 线圈端：两根引出线 `A`、`B`；
- 线圈两端并联一个阻尼电阻 `R_p`；
- 现场已有屏蔽双绞线和 SMB 同轴线材；
- 采集通道：`PXI2Slot2/ai31`；
- 采集方式：优先使用差分输入。

本文档只定义电气拓扑和调试顺序，不定义 PXIe-4309 接线盒的具体针脚号。实际针脚必须按照 PXIe-4309、接线盒和电缆手册确认。

## 2. 推荐方案

推荐让**屏蔽双绞线承担线圈到采集卡的差分信号传输**，让 `SMB` 线只作为经过确认的短距离转接或测试线，不把一根普通 SMB 同轴线当作完整的差分屏蔽双绞线使用。

总体连接关系：

```text
                 ┌────────────── R_p ──────────────┐
                 │                                  │
线圈端 A ────────●                                  ●──────── 双绞线芯线 1 ─── ai31+
                 │                                  │
线圈端 B ────────●──────────────────────────────────●──────── 双绞线芯线 2 ─── ai31-
                                                                                  │
屏蔽双绞线屏蔽层 ────────────────────────────────────────────────────────────────┘
                                                                      单端接屏蔽/参考地
```

更准确地说，`R_p` 应直接并联在线圈 A、B 两端：

```text
                         线圈
                 A o────((((((────o B
                   │              │
                   └──── R_p ─────┘
```

线圈和 `R_p` 组成一个接收端网络，双绞线从这两个节点取差分电压：

```text
线圈 A / R_p 节点 ───── twisted pair conductor 1 ───── ai31+
线圈 B / R_p 节点 ───── twisted pair conductor 2 ───── ai31-
屏蔽层 ─────────────── 只在采集卡端接屏蔽地/参考端
```

## 3. 并联电阻 `R_p` 的安装

### 3.1 安装位置

`R_p` 应尽量安装在线框接线盒或线圈引出端附近，而不是安装在很长的电缆末端。这样可以使线圈、阻尼电阻和最初的引出节点形成明确的电路。

推荐结构：

```text
线圈 A ── 接线盒节点 A ── 双绞线芯线 1
              │
              └──── R_p ────┐
                            │
线圈 B ── 接线盒节点 B ──────┴── 双绞线芯线 2
```

### 3.2 电阻要求

- 使用金属膜或其他低寄生、非绕线电阻；
- 电阻值必须根据线圈电感、分布电容、目标频率范围和所需阻尼确定；
- 未测量线圈参数前，不应把某个经验值当作最终值；
- 电阻功率按线圈感应电压计算，并保留足够裕量；
- 电阻本体和焊点应有绝缘及机械固定；
- 不要把 `R_p` 串在线圈回路中，本文要求的是并联。

并联电阻的作用是降低线圈谐振 `Q`、减小尖峰和改善频响。它也会降低接收电压，因此需要在“阻尼”和“灵敏度”之间实测折中。

## 4. 屏蔽双绞线接法

### 4.1 线对分配

选择一对相互绞合的芯线：

```text
芯线 1：线圈 A / R_p 节点 → ai31+
芯线 2：线圈 B / R_p 节点 → ai31-
```

不要把两根不相邻、不绞合的芯线当作差分线使用。线对应全程保持绞合，减少外部磁场对差分回路的耦合。

### 4.2 屏蔽层

推荐屏蔽层只在采集卡端单点连接到接线盒规定的屏蔽地或参考端：

```text
线圈端：屏蔽层悬空并绝缘
采集卡端：屏蔽层 → 接线盒规定的屏蔽/机壳地
```

不要在未经确认时把屏蔽层同时连接到：

- 线圈 B；
- `ai31-`；
- PXI 模拟地；
- 功放功率负端；
- 发射线圈或功率电缆屏蔽层。

是否将屏蔽层接模拟参考地，必须按照实际接线盒的端接方式执行。屏蔽层的连接错误可能形成地环路或把发射电流引入 AI 回路。

## 5. SMB 线材的使用方式

### 5.1 首选：SMB 不放在线圈差分主链路中

如果 PXIe-4309 接线盒提供螺钉端子、弹簧端子或差分端子，推荐：

```text
线圈 A ── 屏蔽双绞线芯线 1 ── ai31+
线圈 B ── 屏蔽双绞线芯线 2 ── ai31-
屏蔽层 ────────────────────── 采集卡端单点接屏蔽地
```

此时 SMB 线作为备用测试线，不参与正式接线。原因是单根 SMB 同轴线只有：

- 中心导体；
- 外部屏蔽层。

它不能同时提供“差分正端、差分负端、独立屏蔽层”这三个独立导体。

### 5.2 如果必须使用一根 SMB 线

只有在确认 `ai31` 可以接受这种单端连接、并且实验室短线调试允许的情况下，才采用：

```text
线圈 A / R_p 节点 ───── SMB 中心导体 ───── ai31+
线圈 B / R_p 节点 ───── SMB 屏蔽层   ───── ai31-
```

这种接法的缺点是：SMB 屏蔽层同时承担负端回路和屏蔽功能，不能再把它当作独立屏蔽层。它不如真正的屏蔽双绞线差分接法，容易产生共模、地环路和屏蔽电流问题。

如果采用该方案：

- SMB 线应尽量短；
- 中心导体接 `ai31+`，屏蔽层接 `ai31-`；
- 不要再把 SMB 屏蔽层额外接到其他地；
- 必须确认 PXIe-4309 的该输入配置允许这种参考关系；
- 先断开功放，仅采集环境噪声和线圈感应信号；
- 用万用表确认中心导体与屏蔽层没有短路。

### 5.3 不推荐的做法

- 把 SMB 中心导体接 `ai31+`，但让屏蔽层悬空；
- 把 SMB 屏蔽层同时接 `ai31-`、机壳地和功率地；
- 将 SMB 线剥开后用鳄鱼夹长期夹住编织屏蔽层；
- 把线圈 A、线圈 B 分别接到两个不同的同轴屏蔽层，却未确认两通道同步和接地关系；
- 把接收线圈的任意一端直接接到功放输出端。

## 6. 推荐的物理施工结构

建议制作一个小型线圈接线盒，而不是把 `R_p` 和线圈细线悬空焊接：

```text
┌──────────────────────── 线圈接线盒 ────────────────────────┐
│                                                            │
│  线圈 A ──端子 A──●──────── 双绞线芯线 1 ───────────────┐  │
│                   │                                    │  │
│                   └── R_p ──┐                          │  │
│                              │                          │  │
│  线圈 B ──端子 B──●──────────┴── 双绞线芯线 2 ─────────┘  │
│                                                            │
│  屏蔽双绞线屏蔽层 ──电缆固定夹/屏蔽端子（采集卡端接地）   │
└────────────────────────────────────────────────────────────┘
```

施工要求：

- 线圈引出线先固定在端子上，再由端子连接电阻和双绞线；
- `R_p` 两端焊接在 A、B 节点，不让电阻承担机械拉力；
- 所有裸露焊点套热缩管；
- 接线盒加应变释放，防止线圈移动时拉断细线；
- 接收电缆和发射功率线分开走线，避免长距离平行；
- 接收端电缆尽量短，必要时采用屏蔽金属盒；
- 线圈方向和 A/B 极性要记录，便于后续相位比较。

## 7. 接入 PXIe-4309 前检查

### 7.1 断电检查

在功放主电源、储能电源和发射功率回路均断开时：

1. 用万用表测线圈 A、B 之间的直流电阻，记录线圈电阻和 `R_p` 的并联结果。
2. 确认 `R_p` 确实连接在 A、B 两节点之间。
3. 测量双绞线芯线 1 是否从 A 连续到输入正端。
4. 测量双绞线芯线 2 是否从 B 连续到输入负端。
5. 确认中心导体、双绞线两芯和屏蔽层之间没有意外短路。
6. 确认屏蔽层只在计划位置接地。
7. 确认 `ai31` 的正负端没有接到 AO 或功率输出。

### 7.2 小信号检查

先不要启动功放：

1. 打开 FDEM 上位机的 `ai31` 实时监测。
2. 观察接收线圈环境噪声和零点漂移。
3. 轻微移动接收线缆，观察是否出现明显尖峰或接触不良。
4. 记录不同 `R_p` 值下的噪声、幅值和频响。
5. 如果噪声随触碰屏蔽层显著变化，优先检查接地和屏蔽，不要继续提高发射功率。

### 7.3 低能量验证

首次接入功放时：

- 使用最低可行 `Vset`；
- 发射功率线和接收线保持分离；
- `ai31` 输入前确认不会超过 PXIe-4309 和接线附件允许范围；
- 同时观察独立示波器和上位机实时监测；
- 任何异常尖峰、持续偏置、削顶或输入饱和都应立即停止测试。

## 8. 重要安全边界

- 该接收线圈接线方案不对 FDEM 功放 IGBT 提供保护。
- `ao0/Signal in` 仍必须是严格零直流、无 offset 的正弦波。
- 接收线圈、`ai31` 和 `R_p` 不能连接到功放大功率输出。
- PXIe-4309 的软件量程不等于输入绝对最大值。
- 并联电阻不能替代 AI 输入限幅、隔离和过压保护。
- 未确认差分端接和屏蔽接法前，不要把 AI 负端随意接地。
- 不要在功放带电时插拔线圈或 AI 接线。

## 9. 待确认参数

正式测量前还需要补充：

- 线框线圈匝数、面积、导线电阻和电感；
- 线圈分布电容或自谐振频率；
- `R_p` 的实际阻值、精度和功率；
- PXIe-4309 接线盒型号及 `ai31+`、`ai31-`、屏蔽端的物理针脚；
- `ai31` 的差分/伪差分/单端配置；
- 接收前端是否有增益、限幅、隔离或滤波；
- SMB 线的阻抗、长度和屏蔽结构；
- 目标 FDEM 频率范围及线圈在该范围内的幅相响应。
